### 2. ⚽ Football Transfers Data Analysis
**Dataset:** 42,764 football transfers from 1984 to 2020

Exploratory analysis of football transfer fees over the decades. Key findings:
- Transfer spending increased dramatically from the 80s to the 2010s
- Number of transfers per year grew consistently from 1984 to 2018
- Total money spent on transfers exploded in the 2010s

**Tools:** Python, pandas, matplotlib, seaborn, Jupyter Notebook
